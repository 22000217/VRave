#### VR로 장애인 간접 체험을?!!   
단순 글과 강의 영상이 아닌, 직접 장애인들의 입장이 되어 그들의 어려움을 느껴보세요~   
> 장애인 인식 개선을 위해 제공되는 서비스가 많지않으며, 비효율적임.   
현재 제공되고있는 장애인 간접 체험은 접근성이 낮음.
